Preguntas Teóricas

  1. ¿Qué es HTML?
    HTML es el lenguaje estándar para crear páginas web
  2. ¿Cómo se escriben las instrucciones en HTML?
    Las instrucciones en HTML se escriben mediante etiquetas encerradas entre signos menor y mayor, como por ejemplo: (<etiqueta>...</etiqueta>)
  3. Cito algunos lenguajes utilizados en el lado del cliente
    HTML, CSS, JavaScript
  4. Cito algunos lenguajes utilizados en el lado del servidor
    PHP, Python, Node.js, Ruby, Java
  5. ¿Qué es CSS?
    CSS es un lenguaje que se usa para definir el estilo y presentación de las páginas HTML
  6. ¿Cómo puedo vincular una hoja de estilo a una página Web?
    Se vincula usando la etiqueta <link> dentro del <head> del HTML, por ejemplo: <link rel="stylesheet" href="estilos.css">
  7. ¿Qué es Javascript?
    JavaScript es un lenguaje de programación que se ejecuta en el navegador y permite crear páginas web interactivas
  8. ¿Cuáles son los métodos de envío de un formulario?
    Los métodos principales son: GET y POST
  9. ¿Qué es PHP?
    PHP es un lenguaje de programación del lado del servidor que se utiliza para desarrollar sitios web dinámicos
